О проекте

Проект BSUIR SmartSens представляет собой веб-интерфейс для мониторинга и визуализации данных с датчиков, подключённых к микроконтроллеру ESP8266. Система предназначена для локального или сетевого отображения параметров окружающей среды, таких как температура и влажность, в реальном времени. Интерфейс ориентирован на простоту, адаптивность и расширяемость, что делает его удобным как для демонстрационных целей, так и для практического применения в лабораторных или домашних условиях.

Используемые технологии

На стороне клиента используется HTML, CSS и JavaScript без сторонних фреймворков. Визуализация данных реализована средствами CSS с использованием переменных и градиентных заливок (conic-gradient) для отображения уровня значений в виде кольцевых индикаторов. Вёрстка адаптирована под разные размеры экранов и построена на flexbox-модели.

Инструменты и архитектура

Серверная часть построена на Node.js с использованием Express. Сервер обслуживает статические файлы из директории public, включая HTML-страницы, стили, скрипты и изображения. Все страницы используют общий заголовок, автоматически вставляемый через JavaScript-функцию, что упрощает поддержку и обновление навигации. Структура проекта организована таким образом, чтобы обеспечить максимальную читаемость, модульность и простоту развертывания.

Проект легко расширяется за счёт добавления новых датчиков, страниц или API-интерфейсов. Возможна интеграция с реальными устройствами через Wi-Fi, а также подключение к внешним источникам данных.
